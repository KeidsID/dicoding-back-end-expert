# Register User

```
POST /users
```

Register as a new user.

<Accordion title="Headers">Content-Type : application/json</Accordion>

<Accordion title="Body">
  ```json
  {
    "username": <string>,
    "password": <string>,
    "fullname": <string>
  }
  ```
</Accordion>

<Accordion title="Response">

### 201 Created

### Body

{
  "status": "success",
